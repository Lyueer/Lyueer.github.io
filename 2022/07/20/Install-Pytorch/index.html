    Anaconda

    安装Anaconda
    略
    打开Anaconda命令行
    Anaconda Prompt(Anaconda3)
    输入命令

    conda -n pytorchE0 python=3.9
//pytorchE0是环境名称(随便起) //python=3.9是安装anaconda时安装的python版本

    激活（进入）pytorchE0环境

    conda activate pytorchE0
//进入到pytorchE0环境

    查看环境中的工具包

    pip list
//发现工具包没有pytorch

    PyTorch

    安装pytorch

    我的电脑有独立显卡

    查看CUDA版本，打开cmd，输入

    nvida-smi

    发现只能装CUDA 10.2及以下版本

    进入pytorch官网首页

    查看官网以前的安装版本

    复制命令到Anaconda命令行，安装

    输入命令

    conda install pytorch==1.10.0 torchvision==0.11.0 torchaudio==0.10.0 cudatoolkit=10.2 -c pytorch

    PyCharm

    安装PyCharm
    略
    在pycharm中添加pytorch环境的解释器

    Jupyter

    使用jupyter
    方法一：在anaconda的base环境安装pytorch
    略
    方法二：在anaconda的自己创建的pytorchE0环境安装jupyter
    进入Anaconda Prompt(Anaconda3)
    输入命令

    conda list
//查看需要的包的名称为：ipykernel

    进入环境

    conda activate pytorchE0

    conda list
//发现并没有ipykernel

    python=3.9的输入这个命令

    conda install nb_conda_kernels

    打开jupyter

    jupyter notebook
